Raw color values from the shared theme. Use for data visualization, charts, or non-semantic purposes.
For UI elements, prefer semantic CSS variable colors (`bg-primary`, `text-muted-foreground`, etc.).

Available as Tailwind classes: `text-orange-500`, `bg-neutral-900`, `border-blue-300`, etc.

## Special Values

- **transparent**: `transparent`
- **current**: `currentColor`
- **black**: `#000000`
- **white**: `#FFFFFF`
